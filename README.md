## Getting Started
### install
```
git clone git@github.com:Justin-lu/react-redux-best-practice.git demo
cd demo
npm install
```

### npm script

```shell
# dev start with test/lint
npm run start 

# prodction start with browser-sync server
npm run start:prod

# production build
npm run build

# mock data
npm run start:mock

# run test
npm run test

# generate test cover report
npm run test:cover

```
